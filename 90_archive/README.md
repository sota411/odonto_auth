# Archive

このフォルダは、主要導線から外した成果物を保存する場所です。削除はせず、研究の経緯を後から追えるように残しています。

## 構成

- `all_training_trials/`: v6/v7 の細かい試行、probe、dry-run、検証条件違いです。
- `legacy_preliminary/`: 旧データセット、初期 notebook、v1-v4 の旧実験、3D予備検証図です。
- `stage1_notebook_history/`: v5-v7 の notebook 履歴です。旧実行パスや Colab/Windows 実行ログを含むため、現行導線から外しています。
- `misc_scripts/`: 主要導線には入れなかった補助スクリプトです。
- `review_artifacts/`: PDFレビューや抽出テキストなどの確認用資料です。

## 方針

主要成果だけを第一段階と第二段階に出し、探索や履歴確認が必要なものは archive に寄せています。
